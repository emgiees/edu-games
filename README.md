# Educational Games Online - Full Version

## Contents
- **Accounting Simulator**: 38 scenarios across 10 rounds
- **Finance Simulator**: 37 scenarios across 10 rounds

## Features
- Online multiplayer with session codes
- Real-time teacher dashboard
- Live leaderboards
- Financial statements (Accounting)
- Built-in calculator & formulas (Finance)
- Glossary for both games
- CSV export of results

## Quick Start

1. Upload all files to your GitHub repository (replace existing files)
2. Ensure `firebase-config.js` has your Firebase credentials
3. Enable GitHub Pages if not already enabled
4. Access your site at: https://username.github.io/repo-name/

## How to Use

### As Teacher:
1. Select "Teacher"
2. Choose Accounting or Finance
3. Click "Create New Session"
4. Share the 6-character code with students
5. Monitor progress in real-time
6. Export results to CSV when done

### As Student:
1. Select "Student"
2. Enter your name and company name
3. Enter the session code from teacher
4. Play through all scenarios
5. Check your rank on the leaderboard

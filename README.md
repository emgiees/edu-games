# Educational Games Online Platform

## Quick Start (No coding required!)

### Step 1: Set up Firebase (free, ~5 minutes)
1. Go to https://console.firebase.google.com
2. Click "Create a project" → name it → create
3. Click "Build" → "Realtime Database" → "Create Database"
4. Choose location, select "Start in test mode" → Enable
5. Click gear icon → "Project settings" → "Your apps" → Add web app
6. Copy the config values shown

### Step 2: Edit firebase-config.js
Open `firebase-config.js` in any text editor (Notepad works!) and paste your values.

### Step 3: Upload to GitHub Pages
1. Create a new repository on GitHub
2. Upload ALL files from this folder (including firebase-config.js)
3. Go to Settings → Pages → Enable from main branch
4. Wait 1-2 minutes, then visit your URL!

### Step 4: Use it!
- **Teacher**: Select Teacher → Create Session → Share code
- **Student**: Select Student → Enter code → Play!
